# Bitray

**A Small Utility For Handling Binary Data Written In AssemblyScript**

## About

- AssemblyScript Compatible
- Small And Fast
- Works In Browser And Node
- Zero Dependencies

## Installation

```bash
~ npm install as-bitray --save
```
